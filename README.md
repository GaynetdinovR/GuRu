## Установка и запуск
<a id="getting_started"></a>

- Клонировать проект

```
git clone https://github.com/GaynetdinovR/GuRu.git
```

- Установить зависимости

```
npm install
```

- Запустить локальный сервер 

```
npm start
```

- Открыть в браузере

```
http://localhost:3000/
```
